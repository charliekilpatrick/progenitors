# Davies et al. (2018) ‚Äî Progenitor masses of Type II supernovae

This directory contains a **Python reimplementation** of the analysis from Davies & Beasor (2018), plus the original IDL code and data used to reproduce the same results.

## Citation

**If you use this code in your research, please cite the original paper:**

- **Davies, B., & Beasor, E. R. 2018**, *The initial masses of the red supergiant progenitors to Type II supernovae*, *Monthly Notices of the Royal Astronomical Society*, **474**, 2116.  
  **DOI:** [10.1093/mnras/stx2734](https://doi.org/10.1093/mnras/stx2734)  
  **ADS:** [2018MNRAS.474.2116D](https://ui.adsabs.harvard.edu/abs/2018MNRAS.474.2116D/abstract)  
  **arXiv:** [1709.06116](https://arxiv.org/abs/1709.06116)

The paper derives the initial mass range of red supergiant progenitors to II-P and II-L supernovae from pre-explosion imaging, including bolometric-correction systematics and an IMF fit yielding an upper mass cut-off M_hi ‚âà 19 M‚äô (with finite-sample and M‚ÄìL systematics raising it to ‚âà 25 M‚äô).

## Directory layout

| Path | Description |
|------|--------------|
| **data/** | Input data: progenitor CSV catalogues (`IIPprog_obsdata_2019.csv`, `IIPprog_obsdata_2022.csv`), M‚ÄìL relation (`M-L_STARS.txt`), and mass list for distribution MC (`input_masses.txt`). |
| **legacy/** | Original IDL code (`.pro`) and reference outputs (`.sav`, `.eps`, `.tex`, `.out`). Preserved for algorithm parity and comparison. |

## Python modules (regenerating the same results)

The following modules replicate the IDL logic so that results (with the same inputs and random seed) match the original implementation.

| Module | Purpose |
|--------|--------|
| **histutils** | `opthist()` ‚Äî histogram with fixed bin edges; `outxhist()` ‚Äî bin centers; `wmean()` ‚Äî weighted mean. Matches IDL `histutils.pro` / `Lfunc_obs_v2.pro`. |
| **powerlaw** | `randomp(gamma, n, range_=..., rng=...)` ‚Äî power-law sampling in linear L (pdf ‚àù L^Œ≥). Matches IDL `randomp`. |
| **m_hi_lo_fit** | `m_hi_lo_fit(x, m_lo, m_hi)` ‚Äî IMF-style cumulative mass at fraction *x* (Œì = ‚àí1.35); `fit_m_hi_lo()` ‚Äî fit M_lo, M_hi via `scipy.optimize.curve_fit`. Matches IDL `m_hi_lo_fit.pro`. |
| **io_utils** | `get_data_path()`, `read_obs_csv()` ‚Äî read progenitor CSV; `obs_df_to_struct()` ‚Äî convert to internal struct; `read_ml_relation()` ‚Äî read M‚ÄìL table. Data paths resolve under `data/` and `legacy/`. |
| **lfunc** | Luminosity-function pipeline: `create_Lobs()` ‚Äî build observed L distribution from CSV (MC trials, histogram, optional save to `LPOBS.npz`); `create_modelgrid()` ‚Äî grid over (L_lo, L_hi, Œ≥), power-law samples + errors; `compare_obs_mod()` ‚Äî log-likelihood / œá¬≤, best fit, write `LFUNC_results.tex` and `LUM_CONTOUR.npz`; `mainproc()` ‚Äî run full pipeline. |
| **prog_mc** | Progenitor mass MC: `run_prog_mc()` ‚Äî read CSV and M‚ÄìL, randomize dmod/BC/A/mag (or L_us+dL_us), L‚Üímass via interpolated M‚ÄìL, sort and fit M_lo/M_hi per trial, 2D histogram and confidence levels, best M_min/M_max. Returns dict with `hist2d`, `minax`, `maxax`, `lconf`, `mmin_best`, `mmax_best`, `masses_fit`, `snname`, etc. |
| **distribution** | Mass-distribution modeling: `generate_distribution(m_min, m_max)` ‚Äî discrete IMF grid (Œ≥ = ‚àí1.35); `generate_sample(masses, probs, size)` ‚Äî draw from that grid; `calculate_chi2(input_masses, sim_masses, lims)` ‚Äî œá¬≤ comparison; `run_mass_simulation()` ‚Äî MC over (m_min, m_max) using `data/input_masses.txt`. |
| **plotting** | `plot_lum_contour()`, `plot_mlo_mhi_contour()`, `plot_lum_comp()`, `plot_mass_spec()`, `plot_all_from_compare()` ‚Äî produce EPS figures under `figs/`. Optional dependency: **matplotlib**. |

## Quick run

```python
from progenitors.davies18 import lfunc, prog_mc

# Luminosity-function pipeline (small grid for speed)
BESTFITPARS, LPOBS, LPMOD = lfunc.mainproc(
    obsfile="IIPprog_obsdata_2022.csv",
    NTRIALS_OBS=1000, NTRIALS_MOD=1000,
    NLLO=10, NLHI=10, NGAM=5,
    seed=42,
)

# Progenitor mass MC
result = prog_mc.run_prog_mc(
    obsfile="IIPprog_obsdata_2019.csv",
    MLfile="M-L_STARS.txt",
    ntrials=1000,
    seed=42,
)
# result["mmin_best"], result["mmax_best"], result["hist2d"], ...
```

**Mass-distribution utilities** (same IMF as above):

```python
from progenitors.davies18 import distribution
masses, probs = distribution.generate_distribution(10.0, 15.0)
samples = distribution.generate_sample(masses, probs, 100)
# run_mass_simulation(input_file="input_masses.txt", n_trials=10000) uses data/input_masses.txt
```

To generate figures (requires matplotlib):

```python
from progenitors.davies18 import plotting
plotting.plot_all_from_compare()  # uses legacy/LUM_CONTOUR.npz
# For prog_mc figures, pass result dict to plot_mlo_mhi_contour, plot_mass_spec, etc.
```
